ColorScheme

This is a color scheme library designed to allow users to import
.xml files generated from paletton.com,
.ase files from Adobe suite applications,
and images.
Colors are provided as Java objects for use in the processing sketch.
Last update: 08/04/2015

For Processing 3.0

latest version 2.0.0 (released 2015.08.04).

For Processing 2.2.1 and below

archived version 1.0.2.

Installation

Unzip the ColorScheme.zip archive and put the extracted ColorScheme
folder into the libraries folder of your processing sketches.
The reference, examples, and source are included.

Reference

Check out the Javadocs.

Use processing's Examples option in the menu to open the examples.
